# Federation Architecture Pattern IoT & AI Specification
The Federation Architecture Pattern (FAP) on IoT and AI shows how data, in federated ecosystems, can be collected from many devices (sensors), shared securely, brought together in one place, and then can be analyzed with Artificial Intelligence so it can be clearly shown on a dashboard.

## Conformance Language

The key words MUST, MUST NOT, REQUIRED, SHALL, SHALL NOT, SHOULD, SHOULD NOT, RECOMMENDED, MAY, and OPTIONAL in this document are to be interpreted as described in RFC 21191.
